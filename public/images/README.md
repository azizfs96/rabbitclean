# مجلد الصور

## إضافة صورة الجوال

لإضافة صورة الجوال الخاصة بتطبيق رابيت كلين:

1. ضع صورة الجوال في هذا المجلد
2. سمّها `phone-mockup.png` أو `phone-mockup.jpg`
3. افتح ملف `resources/views/landing.blade.php`
4. ابحث عن القسم الذي يحتوي على `<!-- App Screenshot Placeholder -->`
5. استبدل الكود الموجود بالكود التالي:

```html
<img src="{{ asset('images/phone-mockup.png') }}" alt="Rabbit Clean App" class="w-full h-full object-cover">
```

## مواصفات الصورة المثالية

- الأبعاد: 1080 × 2340 بكسل (نسبة 9:19.5)
- الصيغة: PNG أو JPG
- الحجم: أقل من 500 كيلوبايت للأداء الأفضل
- المحتوى: لقطة شاشة من تطبيق رابيت كلين

## نصائح

- تأكد من أن الصورة واضحة وجذابة
- استخدم لقطة شاشة تُظهر الواجهة الرئيسية للتطبيق
- يمكنك استخدام أدوات مثل Figma أو Photoshop لإنشاء mockup احترافي
